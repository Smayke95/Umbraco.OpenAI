# Umbraco.OpenAI
Plugin for Umbraco CMS which integrates OpenAI.

## Configuration
Add configuration to appsettings.json

```
"OpenAI": {
  "ApiKey": "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  "Model": "gpt-3.5-turbo" //optional
}
```
